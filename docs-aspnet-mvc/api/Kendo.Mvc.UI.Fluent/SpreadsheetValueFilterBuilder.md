# Kendo.Mvc.UI.Fluent.SpreadsheetValueFilterBuilder
Defines the fluent API for configuring the SpreadsheetSheetFilterSettingsColumn settings.




## Methods


### Index(System.Double)
The index of the column relative to the filter range.


#### Parameters

##### value `System.Double`
The value that configures the index.





### Values(System.Object[])
The filter values for filters that support multiple values.


#### Parameters

##### value `System.Object[]`
The value that configures the values.
